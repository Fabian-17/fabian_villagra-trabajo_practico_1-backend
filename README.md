# fabian_villagra-trabajo_practico-1-backend.

Instalación de dependencias:

```bash
npm install
```

Configurar variables de entorno:

```bash
PORT= 
DB_NAME= 
DB_USER= 
DB_PASSWORD= 
DB_HOST= 
DB_PORT= 
DB_DIALECT= 
```

Ejecución del servidor:

```bash
npm run start 
```

Ejecución en desarrollo:

```bash
npm run dev
```
